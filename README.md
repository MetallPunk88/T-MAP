T-MAP Scanner

A simple port scanner written in Python. Works in Termux without root.

Created on 07.05.2026 for fast port checking and basic IP address information gathering. Useful for pentesters, network engineers and just the curious.

What it does:

1) Scans 23 popular ports at once

2) Checks any single port

3) Shows country, city, ISP and map link for any IP

Why you need it:

When you gotta quickly check which ports are open on a server or find out where a suspicious IP is coming from. Everything in one simple menu without hassle.

Installation:

1) git clone https://github.com/MetallPunk88/t-map.git

2) cd t-map

3) python t-map.py

Requirements:

1) Python 3
2) Termux/Windows or any Linux
